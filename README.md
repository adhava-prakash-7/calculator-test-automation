# Calculator Test Automation

This project demonstrates basic automation testing of a calculator web application using **Java**, **Selenium WebDriver**, and **TestNG**.

## Project Description
The purpose of this project is to validate core calculator functionalities such as:
- Addition
- Subtraction
- Multiplication
- Division

Automation scripts are executed to ensure the calculator produces correct results for different inputs.

## Tools & Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Eclipse IDE
- Chrome Browser
- ChromeDriver

## Test Scenarios Covered
- Verify addition operation with valid inputs
- Verify subtraction operation
- Verify multiplication operation
- Verify division operation
- Validate calculator output display

## How to Run the Project
1. Clone the repository
2. Open the project in Eclipse
3. Configure ChromeDriver
4. Run the TestNG test class
5. View results in the TestNG report

## Note
This is a **practice automation project** created to understand Selenium execution, test scenarios, and result validation.
